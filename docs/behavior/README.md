2. Диаграммы последовательности

Ниже приведены основные сценарии поведения системы.
Каждая диаграмма представлена в двух форматах:

.puml — исходный PlantUML-код

.png — сгенерированное изображение

2.1. Сценарий №1 — Добавление книги

Показывает процесс добавления новой книги пользователем:
форма → контроллер → сервис → база → подтверждение.

Файлы:

sequence/add_book_sequence.puml

sequence/add_book_sequence.png

2.2. Сценарий №2 — Добавление сессии чтения

Показывает:

добавление сессии

автоматический пересчёт прогресса книги
отображение обновлённого прогресса.

Файлы:

sequence/add_session_sequence.puml

sequence/add_session_sequence.png

2.3. Сценарий №3 — Завершение чтения книги

Показывает:

изменение статуса книги на «Прочитана»
